#Devstype - dahatdevs

Running Locally
Note

This project uses npm only as a package manager.

Clone the repository:

```
git clone https://github.com/prajwal-dahat0810/devstype.git

```

Navigate to the project directory:

`Run frontend`

```
cd frontend
npm run dev

```

Note

Your Docker Demon should be online
